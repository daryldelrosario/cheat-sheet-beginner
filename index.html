HTML Semantic Tag Reference

Tag Name Description
<header> Header Describes content at top of page.
<nav> Navigation Encapsulates page navigational links.
<main> Main Encapsulates main content of page.
<footer> Footer Includes footer content at bottom of <body>
<section> Section Defines elements in a document with same theme (e.g. chapters, headings)
<article> Article Holds contents that makes sense on its own (e.g. articles, blogs, comments)
<aside> Aside Marks information that may enhance but not required to understand content.
<figure> Figure Encapsulates media (i.e. image, diagram or code snippet)
<figcaption> Figure Caption Normally used within <figure> to group media and description.
<embed> Embed Self closing tag that implements any type of media.
<video> Video Closing tag that adds videos to website.
<audio> Audio Closing tag that adds audio to website.
